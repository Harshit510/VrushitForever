# Engagement Ceremony Portfolio

A responsive Vite + React + TypeScript portfolio website for an engagement ceremony. The first version includes a full ceremony landing experience with a hero, couple story, event details, moment cards, schedule, venue action, and RSVP action.

## Tech Stack

- Vite
- React
- TypeScript
- ESLint

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run preview
```

## Personalization Notes

Update the sample couple names, date, venue, schedule, RSVP email, and gallery imagery in `src/App.tsx` and `src/App.css` when final ceremony details and photos are available.
